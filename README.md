# Discourse Sponsor Plugin

Adds sponsorship support with configurable payment providers.

## Settings

All plugin settings are prefixed with `discourse_sponsor_` to avoid duplicates.

### WeChat Pay

- `discourse_sponsor_wechat_enabled`
- `discourse_sponsor_wechat_app_id`
- `discourse_sponsor_wechat_merchant_id`
- `discourse_sponsor_wechat_private_key`
- `discourse_sponsor_wechat_public_key`
- `discourse_sponsor_wechat_notify_url`

### Alipay

- `discourse_sponsor_alipay_enabled`
- `discourse_sponsor_alipay_app_id`
- `discourse_sponsor_alipay_merchant_id`
- `discourse_sponsor_alipay_private_key`
- `discourse_sponsor_alipay_public_key`
- `discourse_sponsor_alipay_notify_url`
